# TestOps Copilot Frontend

AI-ассистент для автоматизации работы QA-инженера на базе Cloud.ru Evolution Foundation Model.

## Установка и запуск
- Доступ к Cloud.ru Evolution Model API

### Установка
```bash
# Клонирование репозитория
git clone https://github.com/phcrlt/testops.git
cd testops-copilot

# Установка зависимостей
cd testops
npm install
npm run build

cd ..
cd backend
pip install -r requirements.txt

cd ..

запустить start.bat
